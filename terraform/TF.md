## Change Infrastructure

As a change of infrastructure I changed the external port of the nginx container

```
resource "docker_container" "nginx" {
  image = docker_image.nginx.latest
  name  = "tutorial"
  hostname = "learn-terraform-docker"
  ports {
    internal = 80
-   external = 8000
+   external = 8080
  }
}
```

## Define input variables

The configuration of Input Variable

```
variable "container_name" {
  description = "Value of the name for the Docker container"
  type        = string
  default     = "ExampleNginxContainer"
}
```

Applying the terraform config with variable with the command `terraform apply -var "container_name=nginx_container"`

## Define Outputs

The configuration of output

```
output "container_id" {
  description = "ID of the Docker container"
  value       = docker_container.nginx.id
}

output "image_id" {
  description = "ID of the Docker image"

}
```
